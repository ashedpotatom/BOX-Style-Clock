# 3D Digital Box Clock

A minimalist 3D Digital Clock built with React, Three.js, and @react-three/fiber.

## Features
- **3D "BOX" Aesthetic**: Inspired by spacetypegenerator.com.
- **Dynamic Fonts**: Toggle between 'Oi' (Extra Bold) and 'Montserrat' (Light) fonts.
- **Auto-Alignment**: Clock aligns to front every minute (00s) or via the 'SPIN' button.
- **Dark/Light Mode**: Minimalist icon-only toggle with hover effects.
- **Ultra-Tight Layout**: Box dimensions auto-recalibrate for a perfect fit around the text.

## Tech Stack
- React
- Three.js
- @react-three/fiber
- @react-three/drei

## Getting Started
1. Install dependencies: `npm install`
2. Run locally: `npm run dev`
3. Build for production: `npm run build`
